# Account-book-app-
Account Book App is a Flutter-based financial management application designed to help users record and manage daily transactions, income, expenses, debts, and credits. The app provides an intuitive interface, transaction history, financial summaries, secure local storage, and efficient tools for personal and small business accounting.

Account Book App

A comprehensive personal finance and account management application built with Flutter. The app allows users to track income, expenses, debts, credits, and financial transactions with an intuitive and user-friendly interface.

Features
Add, edit, and delete financial transactions.
Track income and expenses.
Manage debts and credits.
Generate financial summaries and reports.
Search and filter transactions.
Secure local data storage.
Responsive design for Android and iOS.
Technologies
Flutter
Dart
SQLite / Hive for local storage
GetX for state management
Purpose

This application helps individuals and small businesses organize their finances, monitor cash flow, and maintain accurate account records efficiently.
